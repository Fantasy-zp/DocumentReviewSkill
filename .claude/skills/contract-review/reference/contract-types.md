# 合同类型及审核关注点

## 一、劳动合同

### 关键条款
- 工作内容与地点
- 劳动报酬（基本工资、绩效、奖金）
- 工作时间与休假
- 社会保险与福利
- 劳动保护
- 合同期限与试用期
- 解除与终止条件

### 重点关注
| 关注点 | 说明 |
|--------|------|
| 试用期 | 期限是否符合法律规定（最长6个月） |
| 竞业限制 | 范围、期限（最长2年）、补偿金 |
| 培训服务期 | 违约金是否超过培训费用 |
| 加班规定 | 是否明确加班认定和补偿方式 |
| 解除条件 | 是否存在违法解除情形 |

### 法规参考
- 《劳动合同法》
- 《劳动法》

---

## 二、销售/采购合同

### 关键条款
- 标的物规格、数量、质量标准
- 价款及支付方式
- 交付时间、地点、方式
- 验收标准与程序
- 质量保证与售后
- 所有权转移
- 违约责任

### 重点关注
| 关注点 | 说明 |
|--------|------|
| 标的物描述 | 是否具体明确，避免争议 |
| 质量标准 | 是否有明确的验收依据 |
| 所有权保留 | 付款前所有权归属 |
| 风险转移 | 货物毁损灭失的风险何时转移 |
| 退换货条件 | 是否合理可行 |

### 法规参考
- 《民法典》合同编·买卖合同章

---

## 三、服务合同

### 关键条款
- 服务内容与范围
- 服务标准与交付物
- 服务期限
- 服务费用及支付
- 验收标准
- 保密义务
- 知识产权归属

### 重点关注
| 关注点 | 说明 |
|--------|------|
| 服务范围 | 是否清晰界定，避免范围蔓延 |
| 交付标准 | 是否有客观可衡量的标准 |
| 变更机制 | 服务内容变更如何处理 |
| IP归属 | 服务成果的知识产权归属 |
| 人员约定 | 服务人员的资质和替换机制 |

---

## 四、租赁合同

### 关键条款
- 租赁物描述与用途
- 租赁期限
- 租金及支付方式
- 押金及退还条件
- 维修责任
- 转租限制
- 解除与续约

### 重点关注
| 关注点 | 说明 |
|--------|------|
| 租赁期限 | 超过20年部分无效 |
| 租金调整 | 是否有涨租条款及上限 |
| 押金退还 | 退还条件和时限是否明确 |
| 装修处理 | 租期届满装修如何处理 |
| 优先购买权 | 是否保留承租人优先购买权 |

### 法规参考
- 《民法典》合同编·租赁合同章

---

## 五、借款合同

### 关键条款
- 借款金额
- 借款用途
- 利率与计息方式
- 还款期限与方式
- 担保条款
- 违约责任
- 提前还款

### 重点关注
| 关注点 | 说明 |
|--------|------|
| 利率合规 | 是否超过法定利率上限（LPR的4倍） |
| 综合成本 | 是否有隐藏的费用和利息 |
| 担保责任 | 担保范围和责任限制 |
| 违约加速到期 | 触发条件是否合理 |
| 还款顺序 | 先本还是先息 |

### 法规参考
- 《民法典》合同编·借款合同章
- 最高法民间借贷司法解释

---

## 六、合作/框架协议

### 关键条款
- 合作模式与范围
- 各方权利义务
- 投入与分工
- 收益分配
- 决策机制
- 退出机制
- 竞业与排他

### 重点关注
| 关注点 | 说明 |
|--------|------|
| 合作范围 | 是否排他，边界是否清晰 |
| 责任分担 | 对外责任如何承担 |
| 利益分配 | 分配机制是否公平合理 |
| 退出机制 | 如何退出，资产如何分割 |
| 知识产权 | 合作产生的IP如何归属 |

---

## 七、技术合同

### 关键条款
- 技术内容与范围
- 技术指标与验收标准
- 技术资料交付
- 技术支持与培训
- 保密义务
- 知识产权
- 技术改进归属

### 重点关注
| 关注点 | 说明 |
|--------|------|
| 技术标准 | 是否有明确可测量的技术指标 |
| 验收程序 | 验收流程和争议处理 |
| IP归属 | 开发成果的权属约定 |
| 后续改进 | 改进技术的归属 |
| 竞业限制 | 是否限制同类技术开发 |

### 法规参考
- 《民法典》合同编·技术合同章
